# 主要内容

## 跨域

### 1. 什么是前端跨域
* 跨域是浏览器为了安全而做出的限制策略
* 浏览器请求必须遵循同源策略：同域名、同端口、同协议

### 2. 怎么解决前端跨域
* CORS跨域
* JSONP跨域
* 代理跨域
